# MT4 
Desde Mt4 podemos acceder al metaeditor. 
1.- New (CTRL+N)
2.- Rellenar y escribir. Seleccionar "Nuevo indicador"
3.- Compilar (F7)
4.- Cargamos (f5 o Start)
